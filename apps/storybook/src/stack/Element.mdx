# Web component

Custom element `<rgrm-stack>` via `@rgrmdesign/rgrm-ds-elements/stack`. Importing the
package registers the element and loads `@rgrmdesign/rgrm-ds-css/stack`.

The element uses **light DOM** so children participate in the document flow and design
tokens apply normally.
