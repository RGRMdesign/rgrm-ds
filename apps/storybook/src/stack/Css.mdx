# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/stack`. Apply `.rgrm-stack` and optional gap
modifiers (`.rgrm-stack--gap-xs` through `.rgrm-stack--gap-xl`).

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
