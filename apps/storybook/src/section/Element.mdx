# Web component

Custom element `<rgrm-section>` via `@rgrmdesign/rgrm-ds-elements/section`. Importing
the package registers the element and loads `@rgrmdesign/rgrm-ds-css/section`.

The element uses **light DOM**: content is wrapped in a `.rgrm-section__container`,
and any child marked `slot="background"` is moved into the background layer, so
design tokens from the document apply normally.
