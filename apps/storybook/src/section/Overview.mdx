# Section

A section is a full-width band on a page. Layout and theming live in
`@rgrmdesign/rgrm-ds-css`; React and the web component are thin wrappers that apply
the same BEM classes and set `data-theme`.

- **Theme** — set via `data-theme`; background and text colour come from the theme
  tokens, and the whole theme cascades to the content.
- **Container** — content sits in a centered, max-width wrapper
  (`--rgrm-core-max-width-main` by default).
- **Spacing** — top and bottom padding are set independently from the
  `--rgrm-scale-section-space-*` tokens.
- **Background** — an optional full-bleed layer for an image or video behind the
  content.

Load `@rgrmdesign/rgrm-ds-tokens` once in your app so the CSS custom properties are
defined.

**Design spec** — <a href={FIGMA_LIBRARY_URL}>View in Figma</a> for anatomy, variants,
and usage notes.

Pick a page in the sidebar under **Wrappers → Section**, or jump directly:

- **React** — [`@rgrmdesign/rgrm-ds-react`](?path=/docs/wrappers-section-react--documentation) for React apps
