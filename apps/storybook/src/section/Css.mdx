# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/section`. Apply `.rgrm-section` with spacing
modifiers, set `data-theme` for the theme, and wrap content in
`.rgrm-section__container`. Add `.rgrm-section__background` for full-bleed media.

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
