# Web component

Custom element `<rgrm-inline>` via `@rgrmdesign/rgrm-ds-elements/inline`. Importing the
package registers the element and loads `@rgrmdesign/rgrm-ds-css/inline`.

The element uses **light DOM** so children participate in the document flow and design
tokens apply normally.
