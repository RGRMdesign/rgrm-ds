# Inline

A horizontal layout primitive. Children are placed in a flex row with optional wrap and
configurable gap from the `--rgrm-scale-space-*` primitive tokens.

Load `@rgrmdesign/rgrm-ds-tokens` once in your app so the CSS custom properties are
defined.

**Design spec** — <a href={FIGMA_LIBRARY_URL}>View in Figma</a> for anatomy, variants,
and usage notes.
