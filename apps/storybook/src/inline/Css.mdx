# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/inline`. Apply `.rgrm-inline` and optional gap
modifiers (`.rgrm-inline--gap-sm`, `.rgrm-inline--gap-md`, `.rgrm-inline--gap-lg`).

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
