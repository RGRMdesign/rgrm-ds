# Root

Root applies document-level design tokens to `<html>` and `<body>`. Unlike other
components, it does not render visible UI — it activates global styles via the
`rgrm-root` class.

- **`html`** — `font-size` from `--rgrm-root-font-size` (sets the rem basis)
- **`html ::selection`** — `background-color` and `color` from
  `--rgrm-root-selection-background-color` and `--rgrm-root-selection-color`
- **`body`** — `background-color`, `color`, `font-family`, `font-weight`,
  `line-height`, and `letter-spacing` from `--rgrm-root-*`

Load `@rgrmdesign/rgrm-ds-tokens` once in your app. Theme switching via
`data-theme` on `<html>` continues to work — root tokens reference the active theme.

**Note:** Root (this component) is not the same as the **Root theme** (`data-theme`
omitted / `theme="root"` on Section), which means the default palette.

## Choose an implementation

Pick a page in the sidebar under **Foundations → Root**, or jump directly:

- **React** — [`@rgrmdesign/rgrm-ds-react`](?path=/docs/foundations-root-react--documentation) — wrap your app in `<Root>`
- **CSS** — [`@rgrmdesign/rgrm-ds-css`](?path=/docs/foundations-root-css--documentation) — import root CSS and call `applyRootDocumentClasses()`
- **Web component** — [`@rgrmdesign/rgrm-ds-elements`](?path=/docs/foundations-root-element--documentation) — place `<rgrm-root>` once in the document
