# Web component

Custom element `<rgrm-button>` via `@rgrmdesign/rgrm-ds-elements/button`. Importing
the package registers the element and loads `@rgrmdesign/rgrm-ds-css/button`.

The element uses **light DOM** (inner `<button>` with BEM classes) so design tokens
from the document apply normally.
