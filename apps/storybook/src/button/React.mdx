# React

React wrapper via `@rgrmdesign/rgrm-ds-react`. Importing the package also loads
`@rgrmdesign/rgrm-ds-css` — no separate stylesheet import is required. The
component renders a native `<button>` and forwards all standard button props
(`onClick`, `type`, …).
