# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/button`. Apply `.rgrm-button`, a
variant modifier (`.rgrm-button--primary`), and wrap the label in
`.rgrm-button__label` on a native `<button>` (or `<a>`).

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
