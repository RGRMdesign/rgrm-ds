# Paragraph

Body typography for the RGRM design system. Visual styles live in
`@rgrmdesign/rgrm-ds-css`; React and the web component are thin wrappers that apply
the same BEM classes.

Load `@rgrmdesign/rgrm-ds-tokens` once in your app so the CSS custom properties are
defined.

**Design spec** — <a href={FIGMA_LIBRARY_URL}>View in Figma</a> for anatomy, variants,
and usage notes.

## Choose an implementation

Pick a page in the sidebar under **Typography → Paragraph**, or jump directly:

- **React** — [`@rgrmdesign/rgrm-ds-react`](?path=/docs/typography-paragraph-react--documentation) for React apps
- **CSS** — [`@rgrmdesign/rgrm-ds-css`](?path=/docs/typography-paragraph-css--documentation) for hand-written HTML or any stack without a wrapper
- **Web component** — [`@rgrmdesign/rgrm-ds-elements`](?path=/docs/typography-paragraph-element--documentation) for vanilla JS, server-rendered HTML, or non-React frameworks

Each documentation page includes install steps, copy-paste examples, and all size
variants.
