# Web component

Custom element `<rgrm-paragraph>` via `@rgrmdesign/rgrm-ds-elements/paragraph`. Importing
the package registers the element and loads `@rgrmdesign/rgrm-ds-css/paragraph`.

The element uses **light DOM** (inner `<p>` with BEM classes) so design tokens from the
document apply normally.
