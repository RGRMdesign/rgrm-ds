# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/paragraph`. Apply `.rgrm-paragraph` and optional
size modifiers (`.rgrm-paragraph--large`, `.rgrm-paragraph--small`).

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
