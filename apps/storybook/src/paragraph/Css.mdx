# CSS

Vanilla HTML via `@rgrmdesign/paragraph-css`. Apply `.rgrm-paragraph` and optional
size modifiers (`.rgrm-paragraph--large`, `.rgrm-paragraph--small`).

Use `@rgrmdesign/paragraph-react` or `@rgrmdesign/paragraph-element` when you prefer a
framework wrapper that applies these classes for you.
