# Badge

Inline label or chip for short text (status, count, category). Visual styles live in
`@rgrmdesign/rgrm-ds-css`; React and the web component are thin wrappers that apply
the same `rgrm-` classes.

Load `@rgrmdesign/rgrm-ds-tokens` once in your app so the CSS custom properties are
defined.

**Design spec** — <a href={FIGMA_LIBRARY_URL}>View in Figma</a> for anatomy, variants,
and usage notes.

## Preview

Default variant — colours, padding and radius come from the `--rgrm-badge-*` tokens.

## Choose an implementation

Pick a page in the sidebar under **Content → Badge**, or jump directly:
