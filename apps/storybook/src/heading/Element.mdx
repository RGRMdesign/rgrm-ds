# Web component

Custom element `<rgrm-heading>` via `@rgrmdesign/rgrm-ds-elements/heading`. Importing
the package registers the element and loads `@rgrmdesign/rgrm-ds-css/heading`.

The element uses **light DOM** (an inner `<h1>`–`<h6>` with BEM classes) so design
tokens from the document apply normally.
