# CSS

Vanilla HTML via `@rgrmdesign/rgrm-ds-css/heading`. Apply `.rgrm-heading` together with
a level modifier (`.rgrm-heading--h1` … `.rgrm-heading--h6`).

Use `@rgrmdesign/rgrm-ds-react` or `@rgrmdesign/rgrm-ds-elements` when you prefer a
framework wrapper that applies these classes for you.
